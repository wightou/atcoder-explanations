# CHANGELOG

## v59

- 共通HTMLテンプレートに全ページ分として入っていたスクリプトを整理。
- MathJax は、本文Markdownに数式区切りが含まれる解説ページ・知識ページでのみ読み込むように変更。
  - 生成HTML上の `\(` / `\)` / `\[` / `\]` のエスケープ修正は維持。
- `problem_title_ja` の自動縮小 JavaScript は、日本語タイトル付き問題カードを表示するコンテスト表・教育系コンテスト一覧でのみ読み込むように変更。
- トップ、問題解説ポータル、通常リスト型コンテストページ、タグページ、知識一覧ページなどには、不要なスクリプトを出力しないようにした。
- v57 の一時対応で使った括弧専用 `span` などは引き続き撤去済み。
- 現在使っていない旧ヘッダー用の補助関数を削除。

## v58

- MathJax 設定の `\(` / `\)` / `\[` / `\]` のエスケープを根本修正。
  - 生成HTML内で JavaScript に `\(` などが正しく渡るようにし、通常テキスト中の半角括弧が MathJax の数式区切りとして誤処理されないようにした。
- v57 の一時対応だった `problem_title_ja` 内の半角括弧専用 `span` を撤去。
- `problem_title_ja` 周りの不要な括弧補正 CSS と古いコメントを整理。
- 日本語タイトルの1行表示・自動縮小・省略表示、問題番号と日本語タイトルの薄青色は維持。


## v57
