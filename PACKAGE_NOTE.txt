このZIPはメタ・スクリプト類のみを含む差分配布です。
問題解説本文・知識記事本文・生成済みHTMLを誤って上書きしないため、次のディレクトリは含めていません。

- explanations/
- knowledge/
- docs/
- __pycache__/

反映方法:
1. ZIPをリポジトリ直下に展開します。
2. 必要なら次を実行して生成済みHTMLを更新します。

```bash
python scripts/build_site.py
```

知識記事のスタブを作る場合は、次を実行します。

```bash
python scripts/create_knowledge_stubs.py
```

Windows由来の `*:Zone.Identifier` ファイルを消す場合は、リポジトリ直下で次を実行します。

```bash
./scripts/delete_zone_identifier.sh
```

注意:
- README_for_GPT.md は作業用メモです。公開向けのトップREADMEではありません。
- 公開サイトのトップ表示に使うREADMEは README.md です。
- このファイルにはバージョン別の変更履歴を書き足さず、パッケージの扱い方だけを残します。
